Cap the tip suggestion at $20

Background

The checkout screen calls tip(total) to suggest a gratuity. It used to return a flat 15%: total * 0.15 — fine for small bills, absurd for large group tabs.

Intuition

The fix turns an unbounded rule into a saturating one. Drag to feel where they diverge.

Old: · New:

Code

tip.js
function tip(total) { return Math.min(total * 0.15, 20); }

Math.min returns the smaller of 15% and the $20 ceiling.
